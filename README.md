<h2>电商后台管理</h2>
<h6>项目介绍</h6>
此项目是基于React的电商网站中台管理系统，为线上商城提供管理平台应用，基本功能包括商品管理、权限管理、订单管理、数据统计等。
<h6>项目架构</h6>
ReactRouter、ReactHooks、Node.js、Ant-design、Express、MySQql
<h6>项目启动</h6>
npm start
